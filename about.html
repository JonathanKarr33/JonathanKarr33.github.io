Qualcomm Internships

I have enjoyed getting to intern at Qualcomm over the past three summers.

Summer of 2021 - I created a dashboard for the audio team to improve the functionality of the KATS (Kymera Automated Test Systems) framework. As a result, the team uses this display on a daily basis. This project utilized Python, HTML, CSS, JavaScript, Jinja, and Bootstrap.

Summer of 2022 - I created unit tests in C# for QACT (Qualcomm Audio Calibration Tool). As a result, the team is able to check code functionality easily prior to a new release.

Summer of 2023 - I transitioned the team's build system CMake. As a result, this allows the code to be run across different platforms. This reduced the build time while improving efficiency for unit test analysis.
