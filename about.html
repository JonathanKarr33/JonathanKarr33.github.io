Qualcomm Internships

I have enjoyed getting to intern at Qualcomm over the past three summers.

During the summer of 2021, I created a dashboard for the audio team to improve the functionality of the KATS (Kymera Automated Test Systems) framework. As a result, the team uses this display on a daily basis. This project utilized Python, HTML, CSS, JavaScript, Jinja, and Bootstrap.

During the summer of 2022, I created unit tests in C# for QACT (Qualcomm Audio Calibration Tool). As a result, the team is able to check code functionality easily prior to a new release.

Currently, I'm using CMake to create a build system that allows for code to be run across different platforms. This allows for a reduced build time while improving efficiency for unit test analysis.
